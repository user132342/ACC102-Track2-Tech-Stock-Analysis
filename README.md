# ACC102 Mini Assignment - Track 2
## UK Retail Business Performance Analysis (2022-2025)

### 1. Project Overview
This project analyses the business performance of the UK retail sector across three core categories (Food, Clothing, Electronics) from 2022 to 2025. The analysis focuses on sales trends, profit margin differences, and the relationship between inventory turnover and sales volume, providing data-driven insights for retail business management and decision-making.

### 2. Target Users
- Students in introductory business/accounting courses (like ACC102), who are learning to apply Python to basic financial and operational analysis
- Small retail business owners or new store managers, who want a simple, reproducible way to explore sales trends, profit margins, and inventory-related insights
- Beginners to data analysis, looking for a clear, no-fuss example of retail-focused performance reporting

### 3. Dataset Information
- **Data Source**: Simulated retail dataset (self-generated for educational and analytical purposes)
- **Time Period**: January 1, 2022 – December 31, 2025
- **Key Variables**: Date, Retail Category, Sales Amount, Profit Margin, Inventory Turnover, Cost of Goods Sold
- **Preprocessing**: Missing value removal, date formatting, and time feature extraction (Year/Quarter)

### 4. Analytical Workflow
1. Generate structured simulated retail data
2. Data cleaning and validation
3. Calculate annual sales, category profit margins, and correlation metrics
4. Create visualisations for trend and performance comparison
5. Export cleaned data for reproducibility

### 5. Key Findings
1. Total retail sales show steady growth across the four-year period.
2. Food retail has the highest average profit margin, while Electronics has the lowest.
3. A positive correlation exists between inventory turnover and sales performance.

### 6. How to Run the Project
1. Install required packages: `pip install -r requirements.txt`
2. Open and run the Jupyter Notebook: `retail_analysis.ipynb`
3. All outputs (charts and cleaned data) will be generated automatically

### 7. Output Files
- `clean_retail_data.csv`: Cleaned and processed retail dataset
- `annual_sales_trend.png`: Annual sales trend chart
- `category_margin.png`: Profit margin comparison by category
- `inventory_sales_corr.png`: Inventory turnover and sales correlation plot

### 8. Limitations & Improvements
- Limitations: Based on simulated data; limited to three retail categories
- Improvements: Integrate real retail datasets; add regional analysis; build interactive dashboards
